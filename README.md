## Requisitos

- Criar banco de dados mysql com as seguintes configurações
```
  username: 'root'
  password: 'root'
  database: 'nave'
  host: '127.0.0.1'
  dialect: 'mysql' 
```

## Scripts Disponíveis

No diretório do projeto, execute:

- `npm install`
- `npm run migrate`
- `npm run seed`
- `npm start`

O projeto está sendo executado na porta 3000.
